# 安装说明

## macOS 安装

### 如果应用显示"已损坏"

macOS 会阻止从网络下载的未签名应用。解决方法：

#### 方法 1：移除隔离属性（推荐）

1. 打开终端（Terminal）
2. 运行以下命令（将路径替换为你的实际路径）：

```bash
# 如果是 DMG 文件
xattr -cr ~/Downloads/员工打卡系统_*.dmg

# 如果是 APP 文件
xattr -cr ~/Downloads/员工打卡系统.app
```

3. 然后双击打开应用

#### 方法 2：右键打开

1. 右键点击应用
2. 选择"打开"
3. 在弹出对话框中点击"打开"

#### 方法 3：系统设置允许

1. 打开"系统设置" → "隐私与安全性"
2. 在"安全性"部分，找到被阻止的应用
3. 点击"仍要打开"

## Windows 安装

1. 下载 `.msi` 或 `.exe` 文件
2. 双击运行安装程序
3. 按照提示完成安装

## 文件说明

- **macOS**: 下载 `.dmg` 文件（磁盘映像）
- **Windows**: 下载 `.msi` 文件（安装程序）或 `.exe` 文件（可执行文件）
